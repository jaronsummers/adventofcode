## `aoc hello [FILE]`

describe the command here

```
USAGE
  $ aoc hello [FILE]

OPTIONS
  -f, --force
  -h, --help       show CLI help
  -n, --name=name  name to print

EXAMPLE
  $ aoc hello
  hello world from ./src/star1.ts!
```

_See code: [src/commands/star1.ts](https://github.com/jaronsummers/2020/blob/v0.0.0/src/commands/hello.ts)_

## `aoc help [COMMAND]`

display help for aoc

```
USAGE
  $ aoc help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```

_See code: [@oclif/plugin-help](https://github.com/oclif/plugin-help/blob/v3.2.0/src/commands/help.ts)_
